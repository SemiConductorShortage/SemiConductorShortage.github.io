"Okay, so where do I start?" Is probably what you're thinking.

Okay, so now the majestic <!DOCTYPE html>

    <!DOCTYPE html>

Now, HTML works just fine without it. Some people have old browsers, and that's just kept there for backwards compatibility

Now use the the <html> tag

One thing you should note is, a Starting tag looks like this <p>, whereas the Closing tag looks like </p>. It is important to close your tags so that your elements dont merge. However, you might need to use an element as a container,a nd I'll be demonstrating how, with the HTML tag.

<!DOCTYPE html>
<html>
<head>


</head>
</html>

In this example, we're using the HTML tag to contain the html. The HTML consists of a Head tag, which we'll be using to import links and stylesheets and other data.

Our page needs a title, lets go to the head tag and add a page title!

<!DOCTYPE html>
<html>
    <head>
    <title>My Page! </title>
    </head>
</html>

Result:

Notice How the Head tag contained the title tag, and the title tag contains the text?

When a tag is inside another tag, its called a Nested Element.

The page is still quite empty, so lets add a nice piece of text. How about: Hello, Internet?

We'll be using the h1 tag. You can reuse the h1 again and again, but to get varying sizes, you have a range from h1 to h6. h2 is smaller than h1, and h3 is smaller than h2, and so on until h6.

We'll be using the body tag for the actual html elements.

<!DOCTYPE html>
<html>
    <head>
    <title>My Page! </title>
    </head>
    
    <body>
    
    </body>
</html>

And now, the h1...

<!DOCTYPE html>
<html>
    <head>
    <title>My Page! </title>
    </head>

    <body>
    <h1>Hello</h1>
    </body>
</html>

Okay, now, what if i wanted something like pastrikk.com/aboutme/ rather than pastrikk.com/aboutme/myhtmlsiteiguess.html

Well, make a folder in the directory of where you want the site to be (like D:/Beans/Hello), and call it aboutme or whatever you want it to look like, and create a file called index.html in there

IT HAS TO BE NAMED INDEX.HTML as its the default for sites. Well, not necessarily has to, but it'll save you from a bunch of configuration later on.

Okay, lets make a demo! Okay, so we're gonna use the href

<!DOCTYPE html>
<html>
    <head>
    <title>My Page! </title>
    </head>

    <body>
    <h1>Hello</h1>
    <a href="aboutme">About Me! </a>
    </body>
</html>
